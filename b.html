Grammaire

  • Passé composé / Imparfait

PC: auxiliaire + participe passé

Imparfait: -ais, -ais, -ait, -ions, -iez, -aient

  • Pronom relatif neutre (1)

Ce qui = what / that which (subject)

Ce que = what (direct object)

Ce dont = what ... about (replace de, e.g. besoin de, parler de)

  • Regret / reproche (SI + PQP + CPassé)

PQP: auxiliaire (à l’imparfait) + participe passé

CPassé: auxiliaire (au conditionnel présent) + participe passé

CPrésent: radical + terminaisons de l’imparfait

  • Pronoms relatifs composés (2)

Lequel / laquelle / lesquels / lesquelles

Utilisé après une préposition autre que « de »

Sur, sous, dans, avec, pour, contre, etc.


Duquel / de laquelle / desquels / desquelles

Avec une préposition composée qui utilise « de »

À côté de, loin de, près de, à l’intérieur de, à cause de, autour de, etc.

  • Double pronom

Ordre : direct, lui/leur, y, en

lui/leur : à personne

y : un lieu

en : nom avec « de »

Caractéristiques d’une société dystopique

  • La propagande contrôle les citoyens d’une société.
  • L’information, la liberté de pensée et les autres libertés ne sont pas permises.
  • Les citoyens adorent une représentation ou une idée centrale.
  • Les citoyens sont sous surveillance.
  • Les citoyens vivent dans un état déshumanisé et ont peur du monde extérieur.
  • L'individualité n’est pas valorisée.
  • La société est une illusion d’utopie.
